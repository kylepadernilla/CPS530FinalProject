Installing Flask on Current Ubuntu

This guide will show you how to install Flask on an Ubuntu system. For reference, the Ubuntu version used to host this project is 18.04.

The first step is to ensure you have Python 3 installed on your system. To check this you can use the following command:
python3 -V
This should output something to the effect of:
Python 3.6.6
If it does not, stop here and first install Python 3.

The next step is to install support for Python 3 virtual environemnts. This is done by running:
sudo apt install python3-venv
To then create a virtual environment, make a new directory and then use a shell command to make a new virtual environment like so:
mkdir my_app
cd my_app
python3 -m venv venv
To then make this virtual environment active, you must enter run:
source venv/bin/activate
Your terminal prompt will now be prepended by:
(venv) $
Now its finally time to install Flask! To do so run:
pip install Flask
Flask should now be successfully installed on your system. To confirm, you may run:
python -m flask --version
Which should return...
Flask X.X.X
Python X.X.X
as well as some additional information.
