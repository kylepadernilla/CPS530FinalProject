Conclusions and Experience

Thoughts on React

Did they perform to your expectations?

A: It performed really well, made it easier to design and create websites as you can create components in react.

Were they difficult to install or configure?

A: The CDN versison to install it is much easier that npm installing all the react files but the latter part would allow you to access more features of react.js.

Was it easy to create the page(s) with it?

A: Yes as it was a front-end framework and one member had experience with previous projects using it.

Was the learning curve steep compared with regular HTML/CSS/Javascript/PHP?

A: No it was much easier as it was basically javascript but more component-based.

Would you have done things differently in retrospect? Did you regret your choice of frameworks?

A: No, we were satisfied with the outcome.

Thoughts on Flask?

Flask is very intuitive with how it implements nothing and everything at the same time. Being able to add any library u need without any limitation is very usefull for complex applications.
