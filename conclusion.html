Thoughts on React

Did they perform to your expectations?

Flask performs much like we expected form a microframework tha focuses only what we need. The ecosystem allows for developers to mix and match various components. The ability to use extensions makes it intuitive to build simple applications.

Installation difficulties?

The microframework doesn't require many dependencies, making installation itself very lightweight. It can be installed with a simple pip install.

Ease of creating webpage

Setting up the project directory for Flask was very easy. The creation of a dynamic backend webpage required some creative such as using an extension for an ORM and a database. Although Flask doesn't support databases natively, the plugins helped reduce the complexity of working with a familiar database (SQLLite).

Is there a steep learning curve?

Flask has a very slight curve due to its simplicity and very Pythonic nature. Perhaps developers can more easily dive into web applications with full-stack frameworks, but the introduction to using Flask is very easy for beginners.

Would you have done things differently in retrospect? Did you regret the choice of frameworks?

No, we were more than thrilled with the inherent simplicity of implementing a basic web page with Flask.
